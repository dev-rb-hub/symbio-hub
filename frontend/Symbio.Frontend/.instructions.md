# Frontend Agent Instructions

This file helps AI agents working in the `frontend/Symbio.Frontend` workspace.

## Focus
- Keep frontend changes inside `frontend/Symbio.Frontend` unless a cross-cutting change is required.
- Do not expose backend database schemas or internal API implementation details in frontend code.
- Prefer UI, routing, and component improvements that respect the existing React/Vite architecture.

## Common tasks
- Use `npm run lint` to validate TypeScript and ESLint rules.
- Use `npm run dev` to run the Vite development server locally.
- Use `npm run build` for production build validation.

## Agent guidance
- Preserve the existing component and page structure.
- Avoid introducing new frontend dependencies unless necessary for a clear feature or bug fix.
- Link to root repository guidelines in `AGENTS.md` when broader repo conventions matter.
